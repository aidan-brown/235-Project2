Extras

  • Filters:

    The filters update the display and list of pokemon everytime one of them changes to give a more up to date list quicker.

  • Buttons:

    The buttons allow the user to navigate through the list of pokemon one by one. These buttons also react to the filters placed upon the list of pokemon.

  • Display:

    The display is updated based on the pokemon the user chooses and it reacts to the type of the pokemon (changes colors and background).
